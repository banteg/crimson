# Rewrite (Zig)

Home for the clean, idiomatic rewrite. Keep it separate from decompilation
artifacts.

## Toolchain

- Minimum Zig version: **0.15.2** (see `build.zig.zon`).
- Build: `zig build`
- Run: `zig build run`

If Zig is not on your PATH, use the repo toolchain:

```bash
../tools/zig/current/zig build
```
